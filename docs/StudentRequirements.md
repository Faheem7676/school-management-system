# Student Module Requirements

## Objective
Manage all students in the school.

---

## Student Information

### Basic Details
- Admission Number (Unique)
- First Name
- Last Name
- Date of Birth
- Gender
- Blood Group
- Aadhaar Number (Optional)
- Profile Photo

---

### Contact Information
- Email
- Mobile Number
- Address
- City
- State
- Country
- PIN Code

---

### Parent Information
- Father Name
- Mother Name
- Parent Mobile
- Parent Email

---

### Academic Information
- Admission Date
- Academic Year
- Class
- Section
- Roll Number

---

### Status
- Active
- Inactive
- Graduated
- Left School

---

## Features

- Add Student
- Update Student
- Delete Student
- Search Student
- Filter Student
- Export Student List
