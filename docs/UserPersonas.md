# User Personas

## Project

School Management System

---

# 1. Super Admin

### Responsibilities

- Manage the entire system
- Create and manage users
- Assign roles and permissions
- Configure school settings
- View all reports

---

# 2. Principal

### Responsibilities

- View school dashboard
- Monitor teachers
- Monitor students
- View attendance reports
- View examination reports
- View fee reports

---

# 3. Teacher

### Responsibilities

- Mark attendance
- Manage student marks
- Upload assignments
- View class timetable

---

# 4. Student

### Responsibilities

- View attendance
- View assignments
- View timetable
- View examination results

---

# 5. Parent

### Responsibilities

- View child's attendance
- View examination results
- View fee status
- Receive school notifications

---

# 6. Accountant

### Responsibilities

- Manage fees
- Generate receipts
- View pending payments
- Generate financial reports
