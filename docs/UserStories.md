# User Stories

## Epic 1 - Authentication

### User Story 1

**As a Super Admin, I want to log in securely so that I can manage the entire school system.**

### Acceptance Criteria

- Username is mandatory.
- Password is mandatory.
- Invalid credentials should display an error.
- Successful login redirects to the dashboard.

---

## Epic 2 - Student Management

### User Story 2

**As an Administrator, I want to add a student so that student information can be stored digitally.**

### Acceptance Criteria

- Admission Number must be unique.
- Student Name is mandatory.
- Parent Name is mandatory.
- Class must be selected.
- Student should be saved successfully.

---

### User Story 3

**As an Administrator, I want to edit student information so that incorrect details can be updated.**

---

### User Story 4

**As an Administrator, I want to search students by admission number or name so that I can quickly locate student records.**

---

## Epic 3 - Attendance

### User Story 5

**As a Teacher, I want to mark attendance so that student attendance records are maintained daily.**

---

## Epic 4 - Examination

### User Story 6

**As a Teacher, I want to enter examination marks so that report cards can be generated.**

---

## Epic 5 - Fees

### User Story 7

**As an Accountant, I want to collect student fees so that fee records remain accurate.**
